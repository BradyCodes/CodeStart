Syllabus

Camp Discription: In this camp, we learn all about the basics of understanding computers, data, and writing code. We're going to create many things, like Bots, Web-Scrapers, and even a couple of videogames too! By the end of the week, campers should have a good understanding of the basics of computer programming, and maybe a little bit of computer science too. This is a great camp for anyone who's curious about learning to code, or for those with an interst in computers!

Objectives:

  • Learn How Computers Work
  • Learn How to use and manipulate data
  • Learn About all of the different data-types
  • Learn About Variables, Arrays, and Dictionaries
  • Learn About Functions and Classes
  • Learn About while and for loops
  • Learn How to create your own games (In different styles such as text based and visual)
  • Events like game-creating/coding compition, some of which have prizes
  • HAVE FUN!!!!

Prerequisites:

  • Basic computer skills, ie:
    • Typing
    • Navigating

  • Computers are not needed, as each student will be provided one for the week
